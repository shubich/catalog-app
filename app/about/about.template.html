Faceted E-Commerce Search

Приложение для поиска мобильных телефонов. Есть страница поиска с многочисленными фасетами. При клике на смартфон - отображает детальную страницу с товаром. Данные по смартфонами были позаимствованы у ozon.ru

An application for searching for mobile phones. There is a search page with numerous facets. When you click on the smartphone - displays a detailed page with the product. Data on smartphones were borrowed from ozon.ru
